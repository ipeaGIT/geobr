# Select data type: 'original' or 'simplified' (default)

Select data type: 'original' or 'simplified' (default)

## Usage

``` r
select_geometry_type(temp_meta, simplified_geometry)
```

## Arguments

- temp_meta:

  A data.frame with the metadata of geobr datasets

- simplified_geometry:

  Logical `TRUE` or `FALSE` indicating whether the function should
  return a dataset with the 'original' geometry or a dataset with
  'simplified' geometry (Defaults to `TRUE`)
